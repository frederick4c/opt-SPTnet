# Agent Prompt

Before working on this project, read the files in `context/`:

- `context/context.md` for the project purpose and thesis framing.
- `context/changes.md` for the major optimized-package changes compared with
  `../SPTnet`.
- `context/plan.md` for active next steps and open questions.
- `context/notes.md` for durable project notes from previous agents.

Keep these files updated as work progresses:

- Add new project knowledge to `context/notes.md`.
- Add or revise next steps in `context/plan.md`.
- Update `context/changes.md` when core SPTnet package functionality or
  optimization behavior changes.
- Update `context/context.md` only when the project framing or thesis context
  changes.

When making code changes, preserve the project goal: reproducible and extensible
SPTnet functionality for the MPhil thesis report in `report/`. Prefer changes
that make workflows easier to reproduce, test, explain, and maintain.
